* Data.IntMap
  #+BEGIN_SRC haskell
  type Prefix = Int
  type Mask = Int

  data IntMap a = 
      Branch !Prefix !Mask 
             !(IntMap a) !(IntMap a)
    | Leaf !Prefix a
    | Empty
  #+END_SRC

* Performance Considerations

* Performance
